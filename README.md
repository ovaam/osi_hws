### Описание программы

Программа использует **именованные каналы (FIFO)** для передачи данных между двумя родственными процессами (родительским и дочерним). Обработка данных выполняется в дочернем процессе, а результаты передаются обратно в родительский процесс.

1. **Родительский процесс**:
   - Читает данные из входного файла.
   - Передает данные через именованный канал `FIFO1`.
   - Получает результаты через именованный канал `FIFO2`.
   - Записывает результаты в выходной файл.

2. **Дочерний процесс**:
   - Читает данные из `FIFO1`.
   - Обрабатывает данные (вычисляет разность символов).
   - Передает результаты через `FIFO2`.

### Именованные каналы

- `FIFO1` — канал для передачи данных от родительского процесса к дочернему.
- `FIFO2` — канал для передачи результатов от дочернего процесса к родительскому.
- Каналы создаются с помощью функции `mkfifo`.

### Формат командной строки

```bash
./program <input1> <input2> <output1> <output2>
```

**Тестовые данные**:
  input1.txt: abcde.
  input2.txt: cdefg.
  output1.txt: ab.
  output2.txt: fg.

(остальные тестовые данные с результатами лежат в папках с соответсвующими названиями testDataForN, для удобства)
